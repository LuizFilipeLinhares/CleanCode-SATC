# Projetos de Exercícios em Python

Este repositório contém diversos exercícios e exemplos de código em **Python**, abordando conceitos de:

- Estruturas de dados (listas, dicionários)
- Funções e boas práticas de nomenclatura
- Programação orientada a objetos (POO)
- Manipulação de arquivos e pastas (PowerShell)
- Operações matemáticas e conversões (temperatura, salário, médias)

Todos os exemplos seguem **boas práticas de nomes e legibilidade**, com variáveis e funções em português e inglês, dependendo do contexto do exercício.
